═══════════════════════════════════════════════════════════════════════════════
                         MODEL & APPROACH LEGEND
═══════════════════════════════════════════════════════════════════════════════

  Visual Encoder    │ ResNet-101 (ImageNet-V2) + FPN + SAFE (MHA cross-attn)
  View Fusion       │ SAENet eq.5 — per-view SAFE then element-wise sum
  Text Encoder      │ Bio-ClinicalBERT (emilyalsentzer/Bio_ClinicalBERT)
  Alignment Loss    │ InfoNCE contrastive (vision <-> language)
  Classifier (text) │ Bio-ClinicalBERT + linear head, 14-class BCE
  Classifier (img)  │ SAENet image backbone + linear head, knowledge distil.
  Pair Mining       │ RadGraph entity-F1 >= 0.3 (FactMM-RAG eq.1)
  Entity Tags       │ Stanza NLP, top-20 per report (PKARG conditioning)
  Retrieval         │ FactMM-RAG dense retriever, FAISS IndexFlatIP, top-1
  Verification      │ CrossModal cross-attention re-ranking (ReportVerifier)
  Generator         │ SciFive-base / Flan-T5-base (T5-base architecture)
  RAG Template      │ FactMM-RAG (NAACL 2025) prompt format
  Entity Cond.      │ PKARG soft-AND (img x report) -> 4 MLP-projected tokens
  Decoding          │ Beam search, n=3, length_penalty=1.2, <= 60 words
  Dataset           │ IU X-Ray (Indiana University Chest X-Ray Collection)
