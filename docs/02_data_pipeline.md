## Creating the Data Split

**Script:** `tools/create_split.py`

```bash
python tools/create_split.py
```

**Logic:**
- Loads `indiana_reports.csv` to get all unique patient UIDs
- Shuffles with fixed seed (`np.random.seed(42)`) for reproducibility
- Splits 70 / 10 / 20 by patient (not by image) to prevent data leakage
- Saves `splits/iu_split.csv`

```
Total patients → shuffle → [0:70%] train | [70:80%] val | [80:100%] test
```

## Dataset Class: `IUXrayMultiViewDataset`

**File:** `data/dataset.py`

### Construction

```python
dataset = IUXrayMultiViewDataset(root_dir="C:/Datasets/IU_Xray", split="train")
# split options: "train", "val", "test"
```

### Internals

**Step 1 — Filter by split:**
```
indiana_reports.csv  ──┐
                        ├── merge on uid ──── filter by split UIDs
indiana_projections.csv┘
```

**Step 2 — Build report text:**
```python
merged["report"] = merged["findings"].fillna("") + " " + merged["impression"].fillna("")
```
Combines `findings` and `impression` into a single string per patient.

**Step 3 — Group by patient (UID):**
Each sample is one patient with:
- A list of image filenames (typically 2: frontal + lateral)
- One combined report string

**Step 4 — `__getitem__` per sample:**
```
For each sample:
  1. Load up to 2 images from disk
  2. If only 1 image exists → duplicate it (always output 2 views)
  3. Stack → (2, 3, 224, 224)
  4. Clean the report text
  5. Return (images_tensor, report_string)
```

### Image Transforms

Applied to every loaded image:

```python
transforms.Compose([
    transforms.Resize((224, 224)),
    transforms.ToTensor(),
    transforms.Normalize(
        mean=[0.485, 0.456, 0.406],   # ImageNet mean (RGB)
        std=[0.229, 0.224, 0.225]     # ImageNet std
    )
])
```

Images are loaded as RGB (`.convert("RGB")`) even though IU X-Ray images are grayscale — this replicates the single channel across 3 channels for compatibility with ResNet101 pretrained on ImageNet.

### Report Cleaning (`clean_report`)

The IU X-Ray dataset contains anonymization placeholders that need to be removed:

| Replaced | Reason |
|----------|--------|
| `xxxx` | De-identified patient/hospital name tokens |
| `x-xxxx` | Same, with separator |
| `xxxx.` | Period-terminated version |
| `x-` | Residual artifact after cleaning |

After replacement, all whitespace is collapsed.

### Return Shape

```python
images: torch.Tensor  # shape (2, 3, 224, 224)  — two views
report: str           # cleaned concatenated findings + impression
```
